You can open and resize windows but not always a good idea to do. The global object in the browser is the window object. Methods of the window object are : confirm, prompt and alert. One way to get information about the user browser is through window navigation though it isn’t always accurate. To get information about the url of the page your on use the window location. A way to locate info about the users screen is the window screen object. Don’t use document.write() to write text to the document. The latest version of html is html5.1. To embed video clips into html use the video element. For audio use the audio element.
